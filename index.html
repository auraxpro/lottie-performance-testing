Project Overview

Objective: Determine the optimal Lottie implementation that achieves 95%+ performance scores on GTmetrix and Google PageSpeed, matching or exceeding the current SVG baseline (~96%).

Problem: Replacing animated SVGs with Lottie animations dropped page scores from ~96% to ~86% (-10 points).

Solution: Test 5 different implementation approaches to find the optimal method.
